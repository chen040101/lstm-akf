# 模型对比

## 1. 固定对比模型

当前固定对比以下模型：

- `LSTM-AKF`
- `AKF-only`
- `LSTM-only`
- `ANN`
- `SVR`
- `AR`
- `DT`
- `KNN`

## 2. 当前行为约定

### `LSTM-AKF`

- 对比脚本只尝试加载外部 checkpoint；
- 期望路径：`Baseline Models/lstm_akf/checkpoints/best.pt`；
- 会自动创建目录，但不会自动训练；
- 如果该文件不存在，就直接跳过，并把原因写入结果文件。

### `AKF-only`

- 不训练；
- 直接使用数据加载阶段生成的 AKF 基线作为预测结果。

### `LSTM-only`

- 会在 `Baseline Models/lstm_only/` 下自动训练或复用已有 `best.pt`；
- 若数据集或配置变化，会重新训练。

### 传统机器学习模型

- `ANN / SVR / AR / DT / KNN` 会自动训练或复用 `artifacts/model.pkl`；
- 如果没有 `scikit-learn`，对应模型会被跳过。

## 3. 默认评价范围

默认比较两类范围：

- 单帧：`--point-frames 8 15`
- 连续区间：`--sequence-horizons 8 15`

统一输出指标：

- `DE`
- `MSE`
- `RMSE`
- `MAE`
- `Hit Rate`
- `Inference Time`

其中：

- `Hit Rate` 按 `DE <= hit_threshold` 统计；
- 当前脚本默认 `hit_threshold = 0.02`；
- 如需调整，可在对比命令中增加 `--hit-threshold`。

## 4. 输出位置

对比结果统一写入：`Baseline Models/`

其中常用内容包括：

- `Baseline Models/<model_name>/`：各模型自身产物；
- `Baseline Models/comparison/summary_results.json`：汇总结果；
- `Baseline Models/comparison/detailed_results.json`：详细结果；
- `Baseline Models/comparison/comparison.csv`：表格结果；
- `Baseline Models/comparison/summaries/*.md`：按范围生成的简要说明；
- `Baseline Models/comparison/*.png`：指标图。

## 5. 常用命令

```bash
conda activate yolo
python scripts/compare_models.py --device cuda --point-frames 8 15 --sequence-horizons 8 15
python scripts/visualize_predictions.py --device cuda --sample-index 0 --num-samples 5 --horizons 1 8 15
```

## 6. 真实画面可视化

可视化脚本会读取验证集样本、各模型产物和原视频，默认按“每张图一个来源”输出两类图片：

- 单帧图：先输出真实点，再依次输出各模型在未来第 `h` 帧的预测点；
- 轨迹图：先输出真实未来轨迹，再依次输出各模型未来轨迹。

输出目录默认是：`outputs/visualizations/`

注意：

- 真实点使用数据集 JSON 中的 `target`；
- 原视频帧定位依赖新数据集 JSON 的 `meta` 字段；
- 如果旧数据集没有 `meta`，请先重新运行 `scripts/build_dataset.py` 构建一次数据集。
- 默认不生成局部放大图；如需放大图，可传入 `--crop-padding 160`。

示例：

```bash
python scripts/visualize_predictions.py ^
  --dataset data/dataset/val ^
  --models-dir "Baseline Models" ^
  --videos-dir data/video ^
  --sample-index 0 ^
  --num-samples 5 ^
  --horizons 1 8 15 ^
  --output-dir outputs/visualizations
```

如果需要把所有来源叠加到一张图，可增加 `--mode overlay`；如果两种都要，可使用 `--mode both`。

## 7. 使用建议

如果你想把自己的主模型权重纳入固定对比：

1. 先把 `best.pt` 放到 `Baseline Models/lstm_akf/checkpoints/`；
2. 再运行 `compare_models.py`；
3. 若没有放置该文件，对比仍会继续，只是不会包含 `LSTM-AKF` 这一项。
