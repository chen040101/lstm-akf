# 操作记录

## 1. 环境

```bash
conda create -n yolo python=3.10
conda activate yolo
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu128
pip install scikit-learn scipy joblib threadpoolctl
```

## 2. 构建数据集

在 `newlstm-akf` 根目录执行：

```bash
python scripts/build_dataset.py --val-ratio 0.2 --split-mode segment
```

常用说明：

- 默认输入目录：`data/video/`
- 默认输出目录：`data/dataset/train/`、`data/dataset/val/`
- 默认会生成：`data/dataset/split_manifest.json`

## 3. 训练主模型

```bash
python scripts/train.py --device cuda --epochs 200 --batch-size 64 --early-stopping-patience 20 --early-stopping-min-delta 0.0
```

说明：

- 默认训练 `lstm_akf`；
- 输出目录默认自动创建到 `outputs/expXXX/`；
- 最优权重保存在 `outputs/expXXX/checkpoints/best.pt`；
- 若若干轮没有提升，会自动早停；
- 若想关闭早停，可设置 `--early-stopping-patience 0`。

## 4. 验证模型

```bash
python scripts/validate.py --device cuda --checkpoint outputs/exp001/checkpoints/best.pt
```

## 5. 模型对比

```bash
python scripts/compare_models.py --device cuda --point-frames 8 15 --sequence-horizons 8 15
```

补充：

- `LSTM-AKF` 不会在对比阶段自动训练；
- 若 `Baseline Models/lstm_akf/checkpoints/best.pt` 不存在，就跳过该模型；
- 其他对比模型会按当前实现自动训练或复用已有产物。
