# 项目背景

## 1. 任务场景

项目面向装甲板中心点预测场景：

- 检测模块负责给出当前最好打击的装甲板；
- 预测模块负责输出未来若干帧中心位置；
- 结果最终服务于提前量计算，而不是只做离线坐标回归。

## 2. 当前任务定义

当前任务已经收敛为：

- 输入：连续历史 `xy` 中心点序列；
- 输出：未来 `1~15` 帧归一化中心点；
- 时间尺度：默认按 `40 FPS` 处理；
- 在线规则：短时漏检可补位，连续漏检超过 `5` 帧则停止当前预测。

## 3. 主要难点

这个任务的难点主要来自：

- YOLO 检测点存在抖动；
- 实际视频里会有短时漏检；
- 目标存在转向、自转、变速等非线性运动；
- 需要同时兼顾稳定性和实时性。

## 4. 当前技术路线

当前代码采用两层结构：

- `AKF`：负责状态估计、去噪、补位和未来基线外推；
- `LSTM-AKF`：负责学习非线性运动，并通过门控融合 AKF 分支和直推分支。

当前神经网络输入仍然只使用 `xy`，速度和加速度由 AKF 内部估计提供。

## 5. 当前项目边界

- 主训练已支持早停；
- 数据集默认按 `segment` 级切分 train/val；
- 对比中的 `LSTM-AKF` 只接受手动放置的 checkpoint，缺失时自动跳过。
